\subsection{\-Detailed \-Description}
\-This is the header file for complementary filter. \-A complementary filter combines the good characteristics of an accelerometer and a gyroscope.\par
 \-On the other hand it filters out bad properties like drift or high sensitivity.

\-A \href{http://www.cbcity.de/das-kalman-filter-einfach-erklaert-teil-1}{\tt \-Kalman \-Filter} is another solution for this problem.\par
 \-The result is a little bit better, but it is much more complex.

\-Because of the limitations of ressources of the used controller and the very time critical application i recommend to use a complementary filter.\par
 \-The sampling rate is about five times higher with a \-Kalman \-Filter.
